# S.Saran - Full-Stack Developer Portfolio

A modern, high-performance personal portfolio website built with **Next.js 15**, **Styled Components**, and **Framer Motion**. Featuring a sleek dark futuristic theme, smooth smooth scrolling, and a fully responsive design.

## ✨ Features
- **Dynamic Hero Section:** Animated typing effects and interactive background.
- **Glassmorphic UI:** Modern design using backdrop filters and neon gradients.
- **Custom Cursor:** Interactive trail effect that reacts to your movement.
- **Responsive Layout:** Optimized for desktop, tablet, and mobile devices.
- **Next.js App Router:** Utilizing the latest React features for speed and SEO.
- **Styled Components:** Fully customized CSS-in-JS for maximum design flexibility.

## 🛠️ Built With
- **Frontend:** [Next.js](https://nextjs.org/), [React](https://reactjs.org/)
- **Styling:** [Styled Components](https://styled-components.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** Custom SVG and Emojis
- **Language:** [TypeScript](https://www.typescriptlang.org/)

## 📦 Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Saran2416/Saran-s-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Saran-s-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure
- `/app`: Next.js App Router pages and global layouts.
- `/components`: Reusable UI components (Hero, About, Projects, etc.).
- `/public`: Static assets like profile images and icons.
- `registry.tsx`: Styled Components SSR configuration.
