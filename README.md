# Portfolio Website

This is a modern personal portfolio website built using Next.js with a dark futuristic theme. The website showcases various sections including an introduction, projects, and a contact form, all styled with styled-components and enhanced with animations.

## Features

- **Dark Futuristic Theme**: The website features a sleek dark theme with glassmorphism effects.
- **Responsive Design**: The layout is fully responsive, ensuring a great experience on all devices.
- **Animated Components**: Various animations are implemented for smooth transitions and engaging user interactions.
- **Sticky Navbar**: A sticky navigation bar with a blur effect for easy access to different sections.
- **Project Showcase**: A dedicated section to display projects with hover animations and 3D effects.
- **Contact Form**: A functional contact form with styled inputs and an animated submit button.

## Technologies Used

- **Next.js**: A React framework for building server-side rendered applications.
- **Styled-components**: For styling the components with a modern approach.
- **Framer Motion**: For adding animations and transitions to the components.

## Getting Started

To get a local copy up and running, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/portfolio-website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio-website
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to see your portfolio in action.
